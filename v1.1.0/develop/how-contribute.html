How to Contribute to SINGA¶

As with any open source project, there are several ways you can help:

  • Join the mailing list and answer other user’s questions.
  • Build Singa by yourself.
  • Report bugs, feature requests and other issues in the issue tracking application.
  • Check SINGA’s development schedule and contribute code by providing patches.
  • Help with the documentation by updating webpages that are lacking or unclear.
