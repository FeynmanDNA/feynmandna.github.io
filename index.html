A Distributed Deep Learning Platform

Get Started
Apache SINGA is an Apache Top Level Project, open source distributed training platform for deep learning and machine learning models

Distributed Learning

Apache SINGA focuses on distributed deep learning by partitioning the model and data onto nodes in a cluster and parallelize the training

AutoML and Model Zoo

Apache SINGA v2.0.0 has AutoML features, a Healthcare model zoo, and facility for porting other models onto SINGA
